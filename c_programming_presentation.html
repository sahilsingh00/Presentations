Strings - Text Manipulation

üìù Strings are Arrays of Characters

üéØ What is a String?

A string is a sequence of characters terminated by a null character '\0'.

üìù Creating Strings

#include <stdio.h> #include <string.h> int main() { // Method 1: Character array char name1[20] = "John"; // Method 2: String literal char name2[] = "Alice"; // Method 3: Character by character char name3[10]; name3[0] = 'B'; name3[1] = 'o'; name3[2] = 'b'; name3[3] = '\0'; // Null terminator printf("Name 1: %s\n", name1); printf("Name 2: %s\n", name2); printf("Name 3: %s\n", name3); return 0; }
üì§ Output:
Name 1: John
Name 2: Alice
Name 3: Bob

Pointers - Memory Addresses

üìç Pointers Store Memory Addresses

üéØ What is a Pointer?

A pointer is a variable that stores the memory address of another variable.

üè† Memory Analogy

Variable = House | Pointer = House Address

üìù Pointer Syntax

#include <stdio.h> int main() { int number = 42; // Regular variable int *ptr; // Pointer declaration ptr = &number; // Store address of number printf("Value of number: %d\n", number); printf("Address of number: %p\n", &number); printf("Value of ptr: %p\n", ptr); printf("Value pointed by ptr: %d\n", *ptr); return 0; }
üîë & = Address of | * = Value at address

Pointers in Action

üéØ Swapping Numbers Using Pointers
#include <stdio.h> // Function to swap two numbers using pointers void swap(int *a, int *b) { int temp = *a; *a = *b; *b = temp; } int main() { int x = 10, y = 20; printf("Before swap:\n"); printf("x = %d, y = %d\n", x, y); swap(&x, &y); // Pass addresses printf("\nAfter swap:\n"); printf("x = %d, y = %d\n", x, y); return 0; }
üì§ Output:
Before swap:
x = 10, y = 20

After swap:
x = 20, y = 10
üí° Tip: Pointers allow functions to modify original variables!

File Handling Example

üìù Student Records Management
#include <stdio.h> int main() { FILE *file; char name[50]; int age; // Writing to file file = fopen("student.txt", "w"); if (file == NULL) { printf("Error opening file!\n"); return 1; } fprintf(file, "John Doe\n"); fprintf(file, "Age: 20\n"); fprintf(file, "Grade: A\n"); fclose(file); printf("Data written to file successfully!\n"); // Reading from file file = fopen("student.txt", "r"); if (file == NULL) { printf("Error opening file!\n"); return 1; } printf("\nReading from file:\n"); char line[100]; while (fgets(line, sizeof(line), file)) { printf("%s", line); } fclose(file); return 0; }
üì§ Output:
Data written to file successfully!

Reading from file:
John Doe
Age: 20
Grade: A
