What is C?

C is a general-purpose, procedural programming language developed by Dennis Ritchie at Bell Labs in 1972.

  • Low-level programming language with high-level features
  • Foundation for many modern programming languages
  • Widely used in system programming and embedded systems
  • Portable across different platforms
  • Efficient and fast execution
2

Why Learn C?

Advantages

  • Understanding of computer fundamentals
  • Memory management control
  • Fast execution speed
  • Extensive standard library
  • Platform independent

Applications

  • Operating Systems (Linux, Windows)
  • Embedded Systems
  • Device Drivers
  • Database Systems
  • Compilers and Interpreters

Input/Output Operations

Output - printf()

printf("Hello World"); printf("Age: %d", age); printf("Price: %.2f", price); printf("Grade: %c", grade);

Input - scanf()

Format Specifiers: %d (int), %f (float), %c (char), %s (string), %lf (double)

Control Structures - Conditional

if-else Statement

int marks = 85; if (marks >= 90) { printf("Grade A"); } else if (marks >= 80) { printf("Grade B"); } else { printf("Grade C"); }

switch Statement

char grade = 'B'; switch (grade) { case 'A': printf("Excellent"); break; case 'B': printf("Good"); break; default: printf("Average"); }

Control Structures - Loops

for Loop

for (int i = 1; i <= 10; i++) { printf("%d ", i); }

while Loop

int i = 1; while (i <= 5) { printf("%d ", i); i++; }

do-while Loop

int i = 1; do { printf("%d ", i); i++; } while (i <= 5);

Key Difference: do-while executes at least once, while for and while may not execute at all

Functions

Functions are reusable blocks of code that perform specific tasks

// Function declaration int add(int a, int b); // Function definition int add(int a, int b) { return a + b; } // Function call int main() { int result = add(5, 3); printf("Sum: %d", result); return 0; }
  • Return type: Data type of value returned
  • Parameters: Input values passed to function
  • Function body: Code that executes when called
11

Arrays

Arrays store multiple values of the same data type in contiguous memory locations

// Array declaration and initialization int numbers[5] = {1, 2, 3, 4, 5}; char name[20] = "John"; // Accessing array elements printf("First number: %d", numbers[0]); numbers[2] = 10; // Modifying element // Loop through array for (int i = 0; i < 5; i++) { printf("%d ", numbers[i]); }
  • Array indexing starts from 0
  • Array size must be specified at declaration
  • Elements are accessed using square brackets
12

Strings

Strings are arrays of characters terminated by null character ('\0')

// String declaration methods char str1[20] = "Hello"; char str2[] = "World"; char str3[10]; // String input/output printf("Enter name: "); scanf("%s", str3); printf("Hello, %s!", str3); // String functions (need #include <string.h>) strlen(str1); // Length of string strcpy(str3, str1); // Copy string strcat(str1, str2); // Concatenate strings strcmp(str1, str2); // Compare strings
13

Pointers

Pointers store memory addresses of variables

int num = 25; int *ptr; // Declare pointer ptr = &num; // Store address of num printf("Value: %d", num); // Prints: 25 printf("Address: %p", &num); // Address of num printf("Pointer: %p", ptr); // Same address printf("Value via pointer: %d", *ptr); // Prints: 25 *ptr = 30; // Change value through pointer printf("New value: %d", num); // Prints: 30
  • & operator: Address-of operator
  • * operator: Dereference operator
  • Pointers enable dynamic memory allocation
14

Structures

Structures group related data of different types together

// Structure definition struct Student { int id; char name[50]; float marks; }; // Structure variable declaration struct Student s1; struct Student s2 = {101, "Alice", 95.5}; // Accessing structure members s1.id = 102; strcpy(s1.name, "Bob"); s1.marks = 87.0; printf("Student: %s, ID: %d, Marks: %.1f", s1.name, s1.id, s1.marks);
15

File Handling

C provides functions to read from and write to files

// File operations FILE *file; // Writing to file file = fopen("data.txt", "w"); if (file != NULL) { fprintf(file, "Hello, File!"); fclose(file); } // Reading from file file = fopen("data.txt", "r"); if (file != NULL) { char buffer[100]; fgets(buffer, 100, file); printf("File content: %s", buffer); fclose(file); }
  • File modes: "r" (read), "w" (write), "a" (append)
  • Always check if file opened successfully
  • Remember to close files after use
16

Dynamic Memory Management

C provides functions for runtime memory allocation and deallocation

// Include stdlib.h for memory functions #include <stdlib.h> // Dynamic memory allocation int *ptr = (int*)malloc(5 * sizeof(int)); if (ptr != NULL) { // Use allocated memory for (int i = 0; i < 5; i++) { ptr[i] = i + 1; } // Free allocated memory free(ptr); ptr = NULL; } // Other memory functions int *arr = (int*)calloc(10, sizeof(int)); // Initialized to 0 arr = (int*)realloc(arr, 20 * sizeof(int)); // Resize

Preprocessor Directives

Preprocessor directives are processed before compilation begins

// Header file inclusion #include <stdio.h> // System header #include "myheader.h" // User-defined header // Macro definitions #define PI 3.14159 #define MAX(a,b) ((a) > (b) ? (a) : (b)) // Conditional compilation #ifdef DEBUG printf("Debug mode enabled"); #endif // Usage double area = PI * radius * radius; int maximum = MAX(10, 20);
  • #include: Include header files
  • #define: Define macros and constants
  • #ifdef, #ifndef: Conditional compilation

Complete Sample Program

A simple program to manage student records

#include <stdio.h> #include <string.h> struct Student { int id; char name[50]; float marks; }; void displayStudent(struct Student s) { printf("ID: %d, Name: %s, Marks: %.2f\n", s.id, s.name, s.marks); } int main() { struct Student students[3]; // Input student data for (int i = 0; i < 3; i++) { printf("Enter student %d details:\n", i+1); printf("ID: "); scanf("%d", &students[i].id); printf("Name: "); scanf("%s", students[i].name); printf("Marks: "); scanf("%f", &students[i].marks); } // Display all students printf("\nStudent Records:\n"); for (int i = 0; i < 3; i++) { displayStudent(students[i]); } return 0; }

C Program Compilation

Understanding how C programs are converted to executable files

Compilation Steps

  • Preprocessing: Handle directives (#include, #define)
  • Compilation: Convert to assembly language
  • Assembly: Convert to machine code
  • Linking: Combine with libraries

GCC Commands

# Compile and run gcc program.c -o program ./program # Compile with warnings gcc -Wall program.c -o program # Debug version gcc -g program.c -o program

Common Programming Errors

Syntax Errors

  • Missing semicolons
  • Unmatched braces
  • Incorrect variable declarations
  • Typos in keywords

Runtime Errors

  • Array index out of bounds
  • Division by zero
  • Memory leaks
  • Null pointer dereference
// Common error examples int arr[5]; arr[10] = 5; // Array bounds error int *ptr = NULL; *ptr = 10; // Null pointer dereference int result = 10 / 0; // Division by zero
