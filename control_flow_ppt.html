Control Flow in Programming

Statements, Blocks, Conditionals, Loops, and Flow Control

Master the fundamental building blocks of program logic and execution flow

Statements and Blocks

Statement: A single line of code that performs an action
int x = 10; // Declaration statement x = x + 5; // Assignment statement printf("Hello"); // Function call statement
Block: A group of statements enclosed in curly braces { }
{ int a = 5; int b = 10; int sum = a + b; printf("Sum: %d", sum); }
Key Point: Blocks create scope - variables declared inside are only accessible within that block.

if-else Statements

Basic if statement: Execute code only if condition is true
int age = 18; if (age >= 18) { printf("You are eligible to vote"); }
if-else statement: Execute different code based on condition
int score = 85; if (score >= 60) { printf("Passed"); } else { printf("Failed"); }

else-if and switch Statements

else-if ladder: Multiple conditions in sequence
int marks = 85; if (marks >= 90) { printf("Grade A"); } else if (marks >= 80) { printf("Grade B"); } else if (marks >= 70) { printf("Grade C"); } else { printf("Grade D"); }
switch statement: Multiple choices based on a single variable
char grade = 'B'; switch(grade) { case 'A': printf("Excellent"); break; case 'B': printf("Good"); break; case 'C': printf("Average"); break; default: printf("Invalid grade"); }

while and for Loops

while loop: Repeats while condition is true
int i = 1; while (i <= 5) { printf("%d ", i); i++; } // Output: 1 2 3 4 5
for loop: Compact loop with initialization, condition, and increment
for (int i = 1; i <= 5; i++) { printf("%d ", i); } // Output: 1 2 3 4 5
Use for loop when: You know the exact number of iterations
Use while loop when: The number of iterations depends on a condition

do-while Loop

do-while loop: Executes at least once, then checks condition
int num; do { printf("Enter a positive number: "); scanf("%d", &num); if (num <= 0) { printf("Invalid input!\n"); } } while (num <= 0);
Key Difference:
  • while loop: Checks condition first, may not execute at all
  • do-while loop: Executes first, then checks condition
Perfect for: Input validation, menu systems, game loops

break and continue Statements

break: Exits the current loop immediately
for (int i = 1; i <= 10; i++) { if (i == 5) { break; // Exit loop when i equals 5 } printf("%d ", i); } // Output: 1 2 3 4
continue: Skips current iteration, continues with next
for (int i = 1; i <= 5; i++) { if (i == 3) { continue; // Skip when i equals 3 } printf("%d ", i); } // Output: 1 2 4 5
Remember: break exits the loop, continue skips to next iteration

goto and Labels

goto statement: Jumps to a labeled statement (use sparingly!)
#include int main() { int choice; menu: printf("1. Add\n2. Subtract\n3. Exit\n"); printf("Enter choice: "); scanf("%d", &choice); if (choice == 1) { printf("Addition selected\n"); goto menu; } else if (choice == 2) { printf("Subtraction selected\n"); goto menu; } else if (choice == 3) { printf("Goodbye!\n"); goto end; } else { printf("Invalid choice!\n"); goto menu; } end: return 0; }
⚠️ Warning: goto makes code hard to read and debug. Use loops and functions instead when possible!
